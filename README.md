#  Airbnb Homepage UI Clone

A clean and visually accurate **Airbnb-style homepage UI**, built using **HTML & CSS**.
This project recreates the core layout of Airbnb, including navigation, search filters, category icons, and property cards.

## Features

* **Sticky Navbar:** Logo, navigation buttons, and profile menu
* **Search Bar Section:** Inputs for destination, check-in, check-out, and guests
* **Category Filter Bar:** Scrollable buttons with icons (Lake, Rooms, Luxe, Earth Homes, etc.)
* **Property Cards:** Grid of listings with images, ratings, prices, and favorite (heart) icons
* **Clean Layout:** Modern, minimal Airbnb-inspired design
* **Floating Help Button:** Always visible at bottom-right

##  Tech Stack

* **HTML5** – Structure and layout
* **CSS3** – Styling, spacing, layout formatting
